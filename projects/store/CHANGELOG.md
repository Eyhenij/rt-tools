# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - Initial Release

### Added

- `BaseStoreService` - Signal-based synchronous state management
- `BaseAsyncStoreService` - Extended store with async operation handling
- Redux DevTools integration with time-travel debugging
- `ModelStatus` enum for request status tracking
- `IStateBase.Async` and `IStateBase.List` interfaces
- `BASE_INITIAL_STATE.ASYNC` constant for initial state
