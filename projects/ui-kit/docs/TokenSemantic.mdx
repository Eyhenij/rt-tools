# Semantic Tokens

Semantic tokens express **intent**, not color. Components consume **only** this tier —
never primitives or HEX (see [Overview](?path=/docs/foundation-design-tokens-overview--docs)
and [Theming](?path=/docs/foundation-theming--docs)).

> All adaptive tokens are declared once via `light-dark(light, dark)` and switch with
> `color-scheme`. Tokens marked **·mat** resolve to Angular Material system colors
> (`var(--mat-sys-*, …)`) when the app has a Material theme.

---

## Background — `--rt-bg-*`

### Background / Base — adaptive page backgrounds

Change with the theme. Used for the main application surfaces.

<Strip
    items={[
        ['base', 'var(--rt-bg-base-base)'],
        ['elevated', 'var(--rt-bg-base-elevated)'],
        ['subtle', 'var(--rt-bg-base-subtle)'],
        ['hover', 'var(--rt-bg-base-hover)'],
        ['active', 'var(--rt-bg-base-active)'],
        ['strong', 'var(--rt-bg-base-strong)'],
        ['emphasis', 'var(--rt-bg-base-emphasis)'],
        ['inverse', 'var(--rt-bg-base-inverse)'],
    ]}
/>

| Token                        | Light             | Dark              | Usage                                                  |
| ---------------------------- | ----------------- | ----------------- | ------------------------------------------------------ |
| `--rt-bg-base-base`          | `#ffffff`         | `#1c1b1e`         | Page background · **mat** surface                      |
| `--rt-bg-base-elevated`      | `#ffffff`         | `#2a292d`         | Cards, modals, dropdowns · **mat** surface-container   |
| `--rt-bg-base-subtle`        | `#f5f6f8`         | `#232226`         | Subdued areas, input backplates                        |
| `--rt-bg-base-hover`         | `#f3f3f3`         | `rgba(#fff, 6%)`  | Row / item hover                                       |
| `--rt-bg-base-active`        | `#eeeeee`         | `rgba(#fff, 10%)` | Pressed / selected                                     |
| `--rt-bg-base-strong`        | `#e0e0e0`         | `#3f3e43`         | Chips, strong neutral fills                            |
| `--rt-bg-base-emphasis`      | `#747474`         | `#a3a3a3`         | Selected controls (active pagination chip)             |
| `--rt-bg-base-emphasis-soft` | `#a3a3a3`         | `#747474`         | Mid-strength neutral fills                             |
| `--rt-bg-base-inverse`       | `#181818`         | `#f3f3f3`         | Inverted blocks (action-bar) · **mat** inverse-surface |
| `--rt-bg-base-inverse-soft`  | `#323033`         | `#eeeeee`         | Snack-bar, hover on inverse                            |
| `--rt-bg-base-overlay`       | `rgba(#000, 32%)` | `rgba(#000, 60%)` | Scrims behind modals / asides                          |

### Background / Static — never change with theme

| Token                  | Value         | Usage                      |
| ---------------------- | ------------- | -------------------------- |
| `--rt-bg-static-light` | `#ffffff`     | Always white               |
| `--rt-bg-static-dark`  | `#181818`     | Always dark (promo blocks) |
| `--rt-bg-static-none`  | `transparent` | Explicit transparent       |

Step structure (example: **primary**):

<Strip
    items={[
        ['subtle', 'var(--rt-bg-accent-primary-subtle)'],
        ['solid', 'var(--rt-bg-accent-primary-solid)'],

        ['disabled', 'var(--rt-bg-accent-primary-disabled)'],

| Step       | Token (primary example)           | Light                   | Dark                      | Usage                                                 |
| ---------- | --------------------------------- | ----------------------- | ------------------------- | ----------------------------------------------------- |
