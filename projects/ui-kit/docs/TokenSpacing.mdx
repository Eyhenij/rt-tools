# Spacing & Layout

All spacing, radii, typography, shadows, motion and z-index are defined as
tokens. Using magic numbers directly is forbidden. Foundations are
**mode-independent** — identical in light and dark themes.

---

## Spacing scale — `--rt-spacing-{N}`

Based on a **4px grid** (plus a 2px micro-step). Named by pixel value, emitted in `rem`:

{[[0, 0], [2, 2], [4, 4], [6, 6], [8, 8], [12, 12], [16, 16], [20, 20], [24, 24], [32, 32], [40, 40], [48, 48], [56, 56], [64, 64]].map(([n, px]) => (

<Bar key={n} n={n} px={px} />
))}

```css
padding: var(--rt-spacing-16) var(--rt-spacing-24);
gap: var(--rt-spacing-8);
```

## Border radius — `--rt-radius-*`

<div style={{ display: 'flex', gap: 16, flexWrap: 'wrap', padding: '12px 0' }}>
    {['xs', 'sm', 'md', 'lg', 'xl', '2xl', 'full'].map((r) => (
        <Rad key={r} r={r} />
    ))}
</div>

| Token              | Value         | Usage                         |
| ------------------ | ------------- | ----------------------------- |
| `--rt-radius-xs`   | `0.25rem`     | Tags, small elements          |
| `--rt-radius-sm`   | `0.5rem`      | Inputs, small buttons         |
| `--rt-radius-md`   | `0.75rem`     | Cards, pagination chips       |
| `--rt-radius-lg`   | `1rem`        | Large cards, modals           |
| `--rt-radius-xl`   | `1.5rem`      | Prominent containers          |
| `--rt-radius-2xl`  | `2rem`        | Hero blocks                   |
| `--rt-radius-full` | `624.9375rem` | Pills, avatars, round buttons |

## Breakpoints

`--rt-breakpoint-{xs,sm,md,lg,xl}` are emitted **for JS reads only** — CSS media
queries cannot consume custom properties. In SCSS use the `$device-*` variables:

```scss
@use 'rt-tools/src/styles/main' as rt;

@include rt.mixins-media-breakpoint-down(rt.$vars-device-md) {
    font-size: 1rem;
}
```
